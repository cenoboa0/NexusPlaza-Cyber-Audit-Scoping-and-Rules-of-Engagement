# Reglas de Participación (Rules of Engagement)

Este documento establece las directrices operativas, cronogramas y protocolos de comunicación para la auditoría de seguridad de Nexus Plaza. La ejecución de este compromiso se realizará de manera 100% remota.

## 1. Cronograma y Plazos
* Inicio del Compromiso: Dos semanas después de la firma del contrato y el acuerdo de confidencialidad (NDA).
* Entrega del Informe Final: Plazo máximo de 60 días tras el inicio de las actividades.

## 2. Modalidad de Ejecución Remota
Para garantizar la seguridad y el control durante la auditoría a distancia, se establecen los siguientes parámetros:
* Conexión Segura: El acceso a la infraestructura interna se realizará exclusivamente a través de una conexión VPN (SSL o IPsec) proporcionada por el cliente.
* Punto de Origen: Todas las actividades de escaneo y pruebas se originarán desde una VLAN interna aislada gestionada por el departamento de IT.
* Seguridad de la Conexión: Se requiere autenticación de múltiples factores (MFA) para el acceso de los auditores a la red corporativa.

## 3. Ventanas de Tiempo para Pruebas
Para minimizar el riesgo de interrupción en los sistemas de producción, se definen los siguientes horarios:
* Pruebas No Invasivas: Durante el horario comercial habitual.
* Pruebas Invasivas y DoS: Únicamente durante las ventanas de mantenimiento (Viernes, Sábado y Domingo de 02:00 AM a 06:00 AM).

## 4. Comunicación y Contactos
El flujo de información se mantendrá mediante informes de actualización semanales y sesiones de teleconferencia programadas.

Contactos Principales del Cliente:
* Director de IT
* Gerente de Almacén
* Gerente de Operaciones

## 5. Manejo de Datos Sensibles y Confidencialidad
Toda la información recolectada durante la auditoría se tratará bajo los términos del acuerdo de confidencialidad firmado. Los datos sensibles descubiertos (como credenciales o información de clientes) se comunicarán únicamente a través de canales de comunicación cifrados definidos previamente con el Director de IT.
